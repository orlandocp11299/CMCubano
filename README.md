# Configuración para Desarrolladores

## Primera vez trabajando en el proyecto

1. Clona el repositorio
2. Copia `local.properties.template` a `local.properties`
3. Edita `local.properties` y añade tu ruta del SDK:
   ```
   sdk.dir=/tu/ruta/al/Android/sdk
   activation.secret.key=CLAVE_DE_DEBUG_COMPARTIDA
   ```
4. Para debug, usa: `CLAVE_DE_DEBUG_COMPARTIDA`
5. Para release, contacta al administrador del proyecto
6. Clave compartida 1234
